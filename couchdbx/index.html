CouchDBX

The one-click CouchDB package for the Mac

CouchDBX is the simplest way to get CouchDB running on your Mac:

  1. Download
  2. Double-Click
  3. Relax

Latest Releases

CouchDBX is now proudly sponsored by Couchio and available on http://www.couch.io/get






Nightly Builds

Feeling adventurous? Lincoln Stoll provides nightly builds of CouchDBX. Be aware that these may be broken and may not work as intended. If you run into problems with a nightly build, go back and try a stable release.

Fine Print

CouchDBX is an unofficial release of Apache CouchDB. It only supports Intel-based Macs running Mac OS X 10.5 Leopard or newer.

CouchDBX bundles Apache CouchDB, ICU, Spidermonkey & Erlang. CouchDBX itself is Open Source under the Apache License 2.0.

CouchDBX is brought to you by Jan Lehnardt, Geoffrey Grosenbach & Jon Gretar.
